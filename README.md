# Calculadora_em_python
Projeto de uma calculadora simples desenvolvida em Python para prática de lógica de programação.

O programa permite que o usuário insira dois números e escolha uma operação matemática para realizar o cálculo.

Operações disponíveis:
- Soma
- Subtração
- Multiplicação
- Divisão

O projeto utiliza estruturas condicionais e entrada de dados pelo usuário para executar as operações e exibir os resultados no terminal.
